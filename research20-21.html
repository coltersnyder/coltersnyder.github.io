Computer Graphics My Resume

In this school year, I was accepted into the Mines Undergraduate Research Fellowship (MURF) for the project Virtual Reality for Engineering Training where I am working on a multiplayer system to allow for student collaboration. This project is primarily done in C# using the Unity Game Engine. This project is being performed under the supervision of Dr. H. Sebnem Duzgun and a poster will be published in April 2021. In the meantime, any request to see code, paper(s) related to the project, or any other information related to the project should be sent to Dr. H. Sebnem Duzgun.
